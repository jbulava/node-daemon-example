Daemon Example
=====================

To install:
	
	npm install
	

Configuring
-----------

- Copy config.sample.json to config.json and supply Twitter credentials

Run Locally
--------------

	grunt
	
Run on Production as a Daemon
-----------------------------

	grunt forever:server1:start
	grunt forever:server1:stop
	grunt forever:server1:restart
